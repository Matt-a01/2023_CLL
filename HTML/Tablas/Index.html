Marcos de trabajo de Java

La palabra inglesa "FRAMEWORK" (marco de trabajo) define, en terminos generales, un conjunto de estantandarizado de conceptos, practicas y criteros para enfocar un tipo de problematica particular que sirve como referencia, para enfrentar y resolver nuevos problemas de indole similar.

En el desarollo de softwere, un framework o infrestroctura digital es una estroctutra conceptual y tecnologica de soporte definido, normalmente con artefactos o modulos de softwere concretos que puede servir de base para la organizacion y desarollo de softwere. Tipicamente hablando, puede incluir, soporte de programas, bibliotecas, y un lenguaje intrepretado, entre otras herramientas, para asi ayuda a desarollar y unir otras herramientas, para ayudar a desarollar y unir componentes de un proyecto
